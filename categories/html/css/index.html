Inline SVG in WordPress Post

Scalable Vector Graphics (SVG) is a trending technology. As SVG is infinitely scalable graphics, it is widely used. There is a problem in using these graphics inside the a WordPress post. Yes, there are two problems with SVG in WordPress post. First thing is that you cannot upload a SVG image into the WordPress uploads and the second one is we cannot use inline SVG in WordPress post.
