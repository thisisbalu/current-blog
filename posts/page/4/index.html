Infinitely Scalable SVG – Scalable Vector Graphics

Scalable Vector Graphics, widely known as SVG is becoming popular now a days. Scalable Vector Graphics is an open standard that was developed by World Wide Web Consortium(W3C) in 1999. SVG is an XML based vector image format for 2D Graphics that has support for interactivity and animation. In the year 2003 it became W3c recommendation. It is text based, which means you can create SVG in simple text file. Simply it is an image that will be written in xml with some of the predefined xml attributes and tags.
